# mmWave-AROF-Dataset
Dataset and benchmark enabling machine learning applications for mmWave analog radio over fiber

Current setup: 

Bandwidth: 1GHz

Modulation Scheme: QPSK (2Gbps)/ 16QAM (4Gbps)/ 32 QAM (5Gbps)/ 64 QAM ((6Gbps))

Carrier frequency: 28GHz/ 29GHz/ 30GHz

Transmission Fiber Length: 1m/5km/10km

PD Received Optical Power: 3dBm/ 5dBm/ 7dBm
